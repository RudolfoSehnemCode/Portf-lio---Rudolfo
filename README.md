# Portfólio Pessoal - Rudolfo Sehnem

## Descrição

Este projeto é um portfólio pessoal desenvolvido para apresentar meu perfil profissional, projetos e informações de contato de forma moderna, responsiva e intuitiva.

O site foi construído com foco em experiência do usuário, utilizando animações, efeitos visuais modernos e integração direta com o WhatsApp para facilitar o contato profissional.

## Demonstração

O portfólio possui:

* Página inicial com apresentação profissional.
* Seção "Sobre Mim".
* Galeria de projetos.
* Formulário de contato integrado ao WhatsApp.
* Navegação suave entre seções.
* Design responsivo para desktop e dispositivos móveis.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* API WhatsApp (wa.me)

## Conceitos Aplicados

* Eventos JavaScript

## Funcionalidades

### Navegação Inteligente

Menu fixo com rolagem suave entre as seções do site.

### Apresentação Profissional

Área de destaque contendo foto, nome e área de atuação.

### Exibição de Projetos

Cards responsivos para apresentação dos trabalhos desenvolvidos.

### Contato via WhatsApp

Formulário que gera automaticamente uma mensagem personalizada e abre uma conversa no WhatsApp.
